🏗️ Components and Overall Structure

Accumulator Register (AC)

16-bit register composed of flip-flops, stores data and connects to common bus

Adder and Logic Circuit

Combinational circuit executing arithmetic and logic microoperations

Three Input Sets:
• 16 from AC outputs
• 16 from Data Register (DR)
• 8 from Input Register (INPR)

Control Gates

Generate control signals: Load (LD), Increment (INR), Clear (CLR)

🎛️ Control Signals of AC

Micro-operations associated with AC

D₀T₅ : AC ← AC ∧ DR
D₁T₅ : AC ← AC + DR
D₂T₅ : AC ← DR
pB₁₁ : AC (0-7) ← INPR
rB₉ : AC ← AC
rB₇ : AC ← shr AC, AC(15) ← E
rB₆ : AC ← shl AC, AC(0) ← E
rB₁₁ : AC ← 0
rB₅ : AC ← AC + 1

CLR (Clear AC)

Boolean Function: CLR(AC) = rB₁₁

Active CLR input of AC when rB₉ control signal is active.

INR (Increment AC)

Boolean Function: INR(AC) = rB₅

Active INR input of AC when rB₅ is active.

LD (Load AC)

Boolean Function: LD(AC) = D₀T₅ + D₁T₅ + D₂T₅ + pB₁₁ + rB₉ + rB₇ + rB₆

Microoperations that activate LD:
• AND operation (AC ← AC ∧ DR)
• ADD operation (AC ← AC + DR)
• Transfer from DR (AC ← DR)
• Transfer from INPR, Complement, Shift operations

🔧 Adder and Logic Circuit Design

16 Identical Stages Design

Each stage corresponds to one bit of the AC register and contains:

  • Full-Adder (FA): For arithmetic operations
  • Control Signal Inputs: D₁T₅(ADD), D₀T₅(AND), D₂T₅(DR), pB₁₁(INPR), rB₉(COM), rB₇(SHR), rB₆(SHL)
// Pseudo-code for one stage of Adder and Logic Circuit Stage_i {
inputs: AC(i), DR(i), INPR(i), C_in, control_signals

FA_sum = FullAdder(AC(i), DR(i), C_in) & D₁T₅
AND_result = AC(i) & DR(i) & D₀T₅
DR_result = DR(i) & D₂T₅
INPR_result = INPR(i) & pB₁₁
COM_result = ~AC(i) & rB₉
SHR_result = (i == 15) ? E : AC(i+1) & rB₇
SHL_result = (i == 0) ? E : AC(i-1) & rB₆

output = (FA_sum | AND_result | DR(i) | INPR(i) | COM_result | SHR_result | SHL_result) & LD(AC)
}
