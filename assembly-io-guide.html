Key Concepts Explained
🚩 What are Flags?

Input Flag: Indicates when a character is ready to be read from input device.

Output Flag: Indicates when the output device is ready to accept a new character.

Polling: The process of repeatedly checking flags until they indicate readiness.

⚡ Why Use Skip Instructions?

SKI (Skip on Input): Efficiently checks if input is ready without complex branching.

SKO (Skip on Output): Efficiently checks if output is ready without complex branching.

Efficiency: Skip instructions make polling loops more compact and faster.

🔄 Program Flow

Polling Loop: Program waits in a loop until I/O device is ready.

Synchronization: Ensures data integrity between CPU and I/O devices.
