🔄 Assembly Block Move Subroutine

Interactive exploration of data movement in assembly language

🎯 Purpose

This subroutine efficiently moves a block of 16 data items from one memory location to another, demonstrating fundamental assembly programming techniques for data manipulation.



📋 Subroutine Code Breakdown

Section Instruction Operand Description
MAIN PROGRAM      BSA MVE — Branch to subroutine
     HEX 100 100 First address of source data
     HEX 200 200 First address of destination data
     DEC -16 -16 Number of items to move
     HLT — Halt program
SUBROUTINE MVE MVE, HEX 0 — Storage for return address
     LDA MVE I MVE I Load source address
     STA PT1 PT1 Store in first pointer
     ISZ MVE MVE Increment return address
     LDA MVE I MVE I Load destination address
     STA PT2 PT2 Store in second pointer
     ISZ MVE MVE Increment return address
     LDA MVE I MVE I Load item count
     STA CTR CTR Store in counter
     ISZ MVE MVE Increment return address
LOOP LOP, LDA PT1 I PT1 I Load source item
     STA PT2 I PT2 I Store in destination
     ISZ PT1 PT1 Increment source pointer
     ISZ PT2 PT2 Increment destination pointer
     ISZ CTR CTR Increment counter
     BUN LOP LOP Repeat 16 times
VARIABLES      BUN MVE I MVE I Return to main program
PT1, — — Source pointer storage
PT2, — — Destination pointer storage
CTR, — — Counter storage

🔧 How It Works

Uses indirect addressing and pointer arithmetic to iterate through source and destination arrays, copying each element while maintaining proper memory addressing.

💡 Key Concepts

Demonstrates subroutine calls, indirect addressing, pointer manipulation, loop control, and parameter passing in assembly language programming.

⚡ Efficiency

Uses increment and skip zero (ISZ) instructions for efficient counting and loop control, minimizing instruction overhead in the copy operation.
