🔍 How it works:

This subroutine reads two characters and packs them into a single word. The first character is shifted left by 8 bits (two SH4 calls) to occupy the upper byte, then the second character is placed in the lower byte. This technique allows storing two ASCII characters in one 16-bit word.

🎯 Program Flow:

This program creates a character input buffer system. It initializes a pointer to buffer location 500₁₆, then continuously calls the IN2 subroutine to get character pairs. Each packed character word is stored at the current pointer location, which is then incremented for the next storage operation. This creates an efficient text input system with automatic buffer management.
