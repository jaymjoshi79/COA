📚What is a BCD Adder?

A Binary-Coded Decimal (BCD) Adder is a specialized digital circuit that performs arithmetic addition on decimal digits represented in 4-bit binary form. Unlike standard binary addition, BCD addition requires special correction logic to maintain valid decimal representation.

In BCD, each decimal digit (0-9) is represented using 4 bits. However, standard binary addition can produce results greater than 9, which are invalid in BCD. The BCD adder automatically detects and corrects these invalid results.

4-bit Binary Adder
→

Input Signals (9 total):

  • A (4 bits): First BCD digit
  • B (4 bits): Second BCD digit
  • Cin (1 bit): Carry from previous stage

Output Signals (5 total):

  • S (4 bits): BCD sum digit
  • Cout (1 bit): Carry to next stage

🔧Correction Logic

Correction is required when the binary sum exceeds 9 or generates a carry. In these cases, 0110 (decimal 6) is added to the intermediate sum.

Correction Equation

C = K + Z8Z4 + Z8Z2

Where C is the carry-out signal that triggers correction
