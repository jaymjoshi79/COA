01Digital Computers & Binary

Why Binary?

Digital computers achieve greater reliability by employing only two states, similar to human logic with binary choices like true/false or yes/no.

0 1 0 1 1 0 0 1
Key Concept: A single binary digit (0 or 1) is called a bit. Information is stored in groups of bits.

2Number Systems

Decimal (Base 10)

Digits: 0-9

Example: 724.5 = 7×10² + 2×10¹ + 4×10⁰ + 5×10⁻¹

Binary (Base 2)

Digits: 0, 1

Example: 101101₂ = 1×2⁵ + 0×2⁴ + 1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 45₁₀

Octal (Base 8)

Digits: 0-7

Example: 736.4₈ = 7×8² + 3×8¹ + 6×8⁰ + 4×8⁻¹ = 478.5₁₀

Hexadecimal (Base 16)

Digits: 0-9, A-F

Example: F3₁₆ = F×16¹ + 3×16⁰ = 15×16 + 3 = 243₁₀

Conversion Methods

Decimal to Binary:
Integer part: Divide by 2, read remainders bottom to top
Fraction part: Multiply by 2, read integer parts top to bottom
Example: 41.6875₁₀ = 101001.1011₂
Binary to Octal: Group bits in sets of 3
Binary to Hex: Group bits in sets of 4
Example: 1010111100010111₂ = 5361.354₈ = AF1.76₁₆

∞Floating-Point Representation

Scientific Notation for Binary

Separates number into mantissa and exponent for wider range

Binary: +1001.11
Floating-point: +0.100111 × 2⁴

Components:
• Mantissa: 0.100111
• Exponent: 4 (100)
Normalization: The most significant digit of the mantissa is non-zero (1 for binary)

Even Parity:
