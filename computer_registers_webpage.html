🖥️ Computer Registers

Essential Storage Elements - High-speed storage locations within the CPU that are crucial for processing information

💾 What are Computer Registers?

Computer registers are high-speed storage locations within the CPU that hold data and addresses needed for quick access during computations and instruction execution. They are the fastest form of storage in a computer system.

Types of Data Stored:

  • Numbers: Used in arithmetic computations
  • Letters: For data processing (binary-coded)
  • Addresses: Memory locations
  • Instructions: Program code being executed

🔢 Number Representations

Registers use different methods to represent signed numbers:

Signed-Magnitude
Sign bit (0=+, 1=-) + magnitude
1's Complement
Complement all bits including sign
2's Complement
Most common method in modern computers

Click on each register to learn more about its function:

Data Register (DR)
16 bits
Holds memory operand - data fetched from or to be stored in memory
Address Register (AR)
12 bits
Holds memory addresses (2¹² = 4096 possible locations)
Accumulator (AC)
16 bits
Primary processor register for arithmetic and logic operations
Instruction Register (IR)
16 bits
Holds the instruction code currently being executed
Program Counter (PC)
12 bits
Holds address of next instruction to be fetched
Temporary Register (TR)
16 bits
Holds temporary data during program execution
Input Register (INPR)
8 bits
Holds input character from external devices
Output Register (OUTR)
8 bits
Holds output character for external devices

⚡ Micro-operations on Registers

Elementary operations performed on register data using Register Transfer Language (RTL):

Register Transfer
R2 ← R1
If (P = 1) then (R2 ← R1)
Arithmetic Operations
R3 ← R1 + R2
R3 ← R1 - R2
R1 ← R1 + 1
Logic Operations
F ← A ∧ B (AND)
F ← A ∨ B (OR)
F ← A ⊕ B (XOR)
F ← A' (Complement)
Shift Operations
R ← shl R (Shift Left)
R ← shr R (Shift Right)
R ← cil R (Circular Left)
R ← ashr R (Arithmetic Right)
