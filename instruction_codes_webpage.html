Instruction Codes

Understanding Computer Operations & Binary Instructions

Introduction to Instruction Codes

A computer instruction is a binary code that specifies a sequence of micro-operations for the computer to perform.

BINARY
What is an Instruction Code?

Instructions are fundamental to how computers process information, dictating the specific operations the machine should execute. Each instruction tells the computer exactly what to do and how to do it.

Example 16-bit Instruction
0010 1100 1010 0001
↑ Opcode ↑ ↑ Address ↑

Key Concepts

1
Binary Foundation

All computer instructions are represented in binary format (0s and 1s) that the processor can understand and execute.

2
Micro-operations

Each instruction triggers a sequence of micro-operations - the smallest executable operations within the processor.

3
Control Unit

The control unit interprets instruction codes and generates appropriate control signals to execute operations.

Components of Instruction Codes

Operation Code (Opcode)

OPCODE
The Heart of Every Instruction

The opcode is a group of bits that defines the specific operation to be performed. Common operations include add, subtract, shift, and complement.

Basic Computer Instruction Set

Memory-Reference Instructions

AND
Logical AND

Performs logical AND between memory word and Accumulator (AC)

Operation: AC ← AC ∧ M[Address]
Example: If AC = 1010 and M[100] = 1100, result = 1000
ADD
Addition

Adds memory word to AC with carry to E flip-flop

Operation: AC ← AC + M[Address]
Note: Overflow sets the E (carry) flip-flop
LDA
Load Accumulator

Loads memory word into the Accumulator

Operation: AC ← M[Address]
Use: Initialize AC with a value from memory
STA
Store Accumulator

Stores AC content into memory location

Operation: M[Address] ← AC
Use: Save AC value to memory
BUN
Branch Unconditionally

Transfers program control to new address

Operation: PC ← Address
Use: Implement loops and jumps
BSA
Branch and Save

Branches to subroutine and saves return address

Operation: M[Address] ← PC, PC ← Address + 1
Use: Implement subroutine calls

Register-Reference Instructions

CLA
Clear AC

Sets Accumulator to zero

CLE
Clear E

Sets E flip-flop to zero

CMA
Complement AC

Complements all bits in AC

INC
Increment AC

Increments AC by one

Input-Output Instructions

INP
Input Character

Inputs character from terminal into AC

OUT
Output Character

Outputs character from AC to printer

SKI
Skip if Input Flag

Skips next instruction if input flag is on

ION
Interrupt On

Turns interrupt facility on
