Program Overview

This assembly program demonstrates a complete interrupt service routine. It begins by saving the processor state, handles input/output operations, and then restores the state before returning to the main program.

  • Saves accumulator and flags to maintain system state
  • Implements input character handling with interrupt
  • Manages output buffer operations
  • Restores original state before exit

Program Flow Analysis

The program follows a structured interrupt handling pattern:

  • Entry (100-104): Interrupt setup and Main program
  • Service (200-NXT): Main interrupt service routine with I/O handling and state saving
  • Exit (EXT): State restoration and return to main program

Key Instructions

  • STA SAC: Store accumulator content
  • CIR: Circular shift right the content into accumulator with E
  • SKI: Skip if input flag is set
  • BUN NXT: Branch unconditionally to NXT
  • ION/IOF: Turn interrupt facility on/off
