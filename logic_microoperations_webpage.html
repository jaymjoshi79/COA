Logic Microoperations

Microoperations are fundamental, low-level operations executed on data stored in registers within a digital computer. They are crucial for manipulating individual bits or portions of binary data within registers and can be used to implement complex machine instructions.

Common Applications

1. Selective Set Operation

Sets bits in register A to '1' where corresponding bits in register B are '1'.

Formula: A ← A ∨ B

Example:
A before: 1010
B (operand): 1100
A after: 1110

2. Selective Clear Operation

Clears bits in register A where corresponding bits in register B are '1'.

Formula: A ← A ∧ B

Example:
A before: 1010
B (operand): 1100
A after: 0010

3. Selective Complement Operation

Complements bits in register A where corresponding bits in register B are '1'.

Formula: A ← A ⊕ B

Example:
A before: 1010
B (operand): 1100
A after: 0110

4. Mask Operation

Clears bits in register A where corresponding bits in register B are '0', isolating specific bit groups.

Formula: A ← A ∧ B

Example:
A before: 1010
B (mask): 1100
A after: 1000

5. Insert Operation

Inserts a new value into specific bit positions using a two-step process: mask to clear, then OR to insert.

Example: Insert 1001 into left four bits:
A before: 0110 1010
After masking: 0000 1010
After insertion: 1001 1010

6. Clear Operation (Comparison)

Compares two numbers and produces all '0's if they are equal using XOR operation.

Formula: A ← A ⊕ B

Example:
A: 1010
B: 1010
Result: 0000 (Equal!)
