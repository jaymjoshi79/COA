Machine Language

Understanding the Computer's Native Tongue

🖥️ What is Machine Language?

A program is a set of instructions designed to direct a computer to perform specific tasks. While we can write programs in various languages, computers can only execute programs in binary form. Machine language is the computer's native tongue - the direct set of instructions that hardware recognizes and executes.

Hexadecimal Program

Location Instruction Description
000 2004 Load accumulator from address 004
001 1005 Add memory word from address 005 to accumulator
002 3006 Store accumulator content to memory location 006
003 7001 Halt computer
004 0053 First operand

006 0000 Store sum here

Symbolic Program

000 LDA 004 // Load first operand into AC
001 ADD 005 // Add second operand to AC
002 STA 006 // Store sum in location 006
003 HLT // Halt computer
004 0053 // First operand
