2 Main Memory

Main Memory serves as the central storage unit, containing instructions and data actively used by the CPU.

RAM and ROM Chips ▼

  • RAM (Random-Access Memory): Permits both read and write operations from any memory location within a fixed time interval, enabling true "random access".
  • ROM (Read-Only Memory): Performs only read operations. Binary information is permanent and cannot be altered during computation. Used for fixed programs and constant tables.
  • PROM: Programmable ROM - can be programmed once after manufacturing.
  • EEPROM: Electrically Erasable PROM - can be erased and reprogrammed electronically.

5 Cache Memory

High-speed buffer memory designed to bridge the speed gap between the fast CPU and slower main memory.

Associative Mapping ▼

A main memory block can be loaded into any free cache location. Offers maximum flexibility but requires complex comparison logic for locating data.

Direct Mapping ▼

Simpler predefined mapping where each memory block maps to only one specific cache line. Reduces search complexity but may cause conflicts.

Set-Associative Mapping ▼

Hybrid approach grouping cache lines into sets. A memory block can reside anywhere within its assigned set, balancing speed and simplicity.

6 Virtual Memory

Memory management concept using auxiliary memory to create the illusion of larger main memory space than physically available.

Address Space vs Memory Space ▼

  • Address Space: The set of logical addresses generated by the running program.
  • Memory Space: The actual physical capacity of main memory available in the system.
