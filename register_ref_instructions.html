⚙️ Register Reference Instructions

Overview

Register-reference instructions are recognized by the control when D₇ = 1 and I = 0. These instructions use bits 0 through 11 of the instruction code to specify one of 12 instructions.

The 12 bits are available in IR(0-11) and are transferred to AR during timing T₂.

Key Concept: The control functions and microoperations are executed with the clock transition associated with timing signal T₃. The Boolean relation D₇I'T₃ is used for convenience, where r designates the symbol for a register-reference instruction. The 12 bits contained in IR(0-11) are denoted by the symbols B₀ through B₁₁.

📝 Important Notes

  • The AC is positive when the sign bit is 0
  • AC(15) = 1 if negative, AC(15) = 0 if positive
  • The HLT instruction clears a start-stop flip-flop S and stops the sequence counter from continuing
  • All register operations are performed during timing signal T₃
