🔄 Algorithm

General Steps

  1. Check the operation (Addition or Subtraction)
  2. Compare the signs of A and B
  3. If signs are same → Add magnitudes (for addition)
  4. If signs differ → Subtract smaller from larger
  5. Assign result sign based on magnitude comparison
  6. If result = 0 → Always make it +0

💡 Key Points

✅ MSB is the sign bit
✅ +0 and −0 are equivalent
✅ Signs determine operation
✅ Zero is always positive
