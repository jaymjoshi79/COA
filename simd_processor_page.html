🖥️ SIMD Array Processor

Single Instruction, Multiple Data Architecture

What is a SIMD Array Processor?

A SIMD array processor is a computer with multiple processing units operating in parallel. The processing units are synchronized to perform the same operation under the control of a common control unit, thus providing a single instruction stream, multiple data stream (SIMD) organization.

Key Features

📡 Broadcast Instructions

Vector instructions are broadcast to all PEs simultaneously

🔄 Parallel Execution

Vector operands distributed to local memories for parallel execution

💾 Local Memory

Each PE has its own local memory for operand storage

🎯 Masking Schemes

Control individual PE status during execution

How It Works: Vector Addition Example

C = A + B

Execution Steps:

  1. Store: Master control unit stores ith components of A and B in local memory Mi
  2. Broadcast: Floating-point add instruction (ci = ai + bi) sent to all PEs
  3. Execute: Addition takes place simultaneously in all local memories
  4. Result: Vector sum produced in one add cycle

Notable Implementation: ILLIAC IV

ILLIAC IV was the best-known SIMD array processor, developed at the University of Illinois and manufactured by the Burroughs Corp. Though no longer in operation, it demonstrated the power of SIMD architecture for specialized computational tasks.

Applications & Limitations

✅ Best Suited For:

  • Numerical problems
  • Vector operations
  • Matrix computations

❌ Less Efficient For:

  • Conventional data processing
  • Irregular computational patterns
  • General-purpose computing
