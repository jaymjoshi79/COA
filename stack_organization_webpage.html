I. Stack Organization Overview

The stack is a fundamental component of computer architecture utilizing a LIFO (Last-In, First-Out) architecture. In the context of computer memory, Stack Memory is a distinct type of memory segmentation, separate from Code Memory and Data Memory.

Stack Memory

LIFO data structure for temporary storage and function calls

Code Memory

Program instructions and executable code

Data Memory

Variables and data storage

III. Stack Instructions: PUSH and POP

PUSH Instruction

Micro-operations:
1. SP ← SP + 1
2. M[SP] ← DR
3. If (SP = 0) then (FULL ← 1)
4. EMPTY ← 0

The PUSH instruction stores data onto the stack by first incrementing the Stack Pointer, then transferring data from the Data Register to the memory location.

POP Instruction

Micro-operations:
1. DR ← M[SP]
2. SP ← SP - 1
3. If (SP = 0) then (EMPTY ← 1)
4. FULL ← 0

The POP instruction retrieves data from the stack by transferring data to the Data Register, then decrementing the Stack Pointer.
